# ✨ Features

## 📝 Question Builder

- Add unlimited MCQs
- Single Questions
- Stimulus Questions
- Roman Number Questions
- Stimulus + Roman Questions
- Live Question Counter
- Search Questions
- Edit Questions
- Delete Questions
- Drag & Reorder Questions

Generate MCQs from

- Story
- Documentation
- Articles
